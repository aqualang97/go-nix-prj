Какое будет значение у переменной x после выполнения программы:
func square(x *float64) {
    *x = *x * *x
}
func main() {
    x := 1.5
    square(&x)
}
----->		2.25

Какая разница между методом и функцией?
	метод доступен через экземпляр структуры
	метод это функция определенной структуры, его нельзя вызвать не обратившись к самой 
	у функции мы пишем параметры, у метода мы ставим эти параметры между ф-ей и ключевым словом func.  Это называется получатель
